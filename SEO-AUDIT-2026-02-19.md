## MEDIUM PRIORITY (POST-LAUNCH OK)

### ğŸŸ¢ MEDIUM-01: Add Schema Markup to Key Pages

**Pages needing schema:**
- `about.html` - Person/Organization schema for Mat
- `services.html` - Service schema
- `pricing.html` - Offer schema
- `assessment-new.html` - WebApplication schema

**Why it matters:** Rich snippets in search results = higher CTR. Helps Google understand page purpose.

**Time:** 20-30 minutes

**Example for about.html:**
```json
{
  "@context": "https://schema.org",
  "@type": "Person",
  "name": "Matthew Keys",
  "jobTitle": "Founder",
  "memberOf": {
    "@type": "Organization",
    "name": "newlens"
  },
  "knowsAbout": ["AI Automation", "Data Analytics", "CRM Systems"],
  "alumniOf": "Member of the British Computer Society"
}
```

### ğŸ”µ LOW-01: Add Canonical Tags

All pages should have self-referencing canonical tags to prevent duplicate content issues.

**Add to each page:**
```html

```

**Why it matters:** Prevents Google from getting confused if test.newlens.uk and newlens.uk both exist simultaneously.

**Time:** 10 minutes (batch operation)
